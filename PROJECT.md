# Phuket SEO — Project Rules

## Stack
- Astro + Cloudflare Pages
- Output: static
- Deploy: GitHub → Cloudflare Pages auto deploy

## Rules
- ห้ามแตะ CSS และ Design
- Meta tag ทุกหน้าต้องผ่าน BaseLayout.astro
- Blog ใช้ Markdown + frontmatter เท่านั้น
- Schema ทุกหน้าต้องมี LocalBusiness

## เวลาเพิ่มบทความ
- สร้างไฟล์ใหม่ใน src/content/blog/
- ใส่ frontmatter ให้ครบ
- ไม่ต้องแตะไฟล์อื่น
